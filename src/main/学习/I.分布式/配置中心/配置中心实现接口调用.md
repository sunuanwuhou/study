# 代码实现

注意这里需要指定value的命名空间，不然默认是application

```java
@ApolloConfigChangeListener(value='')
private void someChangeHandler(ConfigChangeEvent changeEvent) {
    for (String key : changeEvent.changedKeys()) {
        ConfigChange change = changeEvent.getChange(key);
        log.info("Found change - {}", change.toString());
    }
}
```



# 正常的list和map形式


```java
apollo.map = {key1:"value1",key2:"value2",key3:"value3"}
apollo.list = "key1,key2,key3"

```



    @Value("#{${apollo.map}}")
    private Map<String, String> map;
    
    @Value("#{${apollo.list}}")
    private List<String> list;
# mapAndMap形式、mapAndList形式

```java
apollo.mapAndMap = {key1:{subKey1_1:"subValue1_1",subKey1_2:"subValue1_2"},key2:{subKey2_1:"subValue2_1",subKey2_2:"subValue2_2"}}
apollo.mapAndList = {key1:"value1,value2,value3",key2:"value4,value5,value6"}

```

    @Value("#{${apollo.mapAndMap}}")
    private Map<String, Map<String, String>> mapAndMap;
    
    @Value("#{${apollo.mapAndList}}")
    private Map<String, List<String>> mapAndList;
